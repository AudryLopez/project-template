# Next Patterns Template

A Next.js 15 template with TypeScript, Tailwind CSS, ESLint, and Prettier pre-configured.

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development
pnpm dev
```

## Setup with Claude

After cloning, open Claude and say:

```
initial setup
```

Claude will generate all utilities in `src/lib/`.

## Project Structure

```
src/
├── app/              # Next.js App Router
├── components/ui/    # Reusable UI components
├── hooks/            # Custom React hooks
├── lib/              # Utilities (generated by Claude)
├── stores/           # Zustand stores
└── types/            # TypeScript types
```

## Scripts

```bash
pnpm dev          # Start dev server
pnpm build        # Production build
pnpm lint         # Run ESLint
pnpm lint:fix     # Fix ESLint errors
pnpm format       # Format with Prettier
pnpm type-check   # TypeScript check
```
